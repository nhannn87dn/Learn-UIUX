## 💛 Tài nguyên

### 💥 Font chữ

Sử dụng hệ thống Google Font có sẵn

### 💥 Hình ảnh

Sử dụng plugin `Unsplash` để nhúng ảnh miễn phí

### 💥 Content

Sử dụng một số plugin cho việc tạo content mẫu:

- `Content Reel`
- `Lorem Ipsum`

### 💥 Icons

Sử dụng một số plugin cho việc tạo icon:

- `Feather Icons`
- `Font Awesome Icons`
